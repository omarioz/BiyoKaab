# PWA Icons

This directory should contain the following icon files for the Progressive Web App:

- `pwa-192x192.png` - 192x192 pixels (required)
- `pwa-512x512.png` - 512x512 pixels (required)
- `apple-touch-icon.png` - 180x180 pixels (for iOS)
- `favicon.ico` - Standard favicon
- `mask-icon.svg` - SVG mask icon (optional)

You can generate these icons using online tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Or create them manually using image editing software.
